# Random Quote Generator

A simple and responsive Random Quote Generator built using **HTML**, **CSS**, and **JavaScript**. The application fetches random quotes from an API and displays them with the author's name.

## Features

* Generate a new quote with a single click
* Fetches quotes from a public API
* Clean and responsive UI
* Loading state while fetching data
* Error handling for failed API requests
* Modern dark-themed design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Fetch API
* Async/Await

## How It Works

1. User clicks the **Generate Now** button.
2. JavaScript sends a request to the quote API.
3. The quote and author are extracted from the response.
4. The UI updates dynamically without refreshing the page.
